## Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/7ommyShelby/TailwinDJS.git
   ```

2. Navigate to the project folder:

   ```bash
   cd your-repo
   ```

3. Open the `index.html` file in your browser:

   ```bash
   open index.html
   ```
